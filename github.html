Github

Github é um das plataformas de hospedagem de arquivos que utiliza o sistema de controle de versão git. Ele permite que qualquer pessoa cadastrada no site seja capaz de contribuir para qualquer projeto privado e/ou Open Source. Além de permitir a hospedagem de sites gratuitamente, como este, por exemplo.




Comandos

git init
Inicia um novo repositório.

git status
Verifica o status atual do projeto.

git add .
Adiciona todos os arquivos/modificações.

git commit -m "mensagem"
Comita todos os arquivos e modificações feitas.

git pull
Atualiza o repositório local de acordo com o remoto.

git push -u origin master
Enviar todos os arquivos e modificações para o repositório remoto. O primeiro push deve conter o nome do repositório remoto e da branch.

git push
Usado após o primeiro push.

git log
Exibe o histórico.

git branch nomedabranch
Cria uma nova ramificação.

git checkout -b nomedabranch
Troca para uma ramificação existente.

git checkout master
Retorna para a branch principal.

git merge nomedabranch
Une a ramificação criada com a principal.

git branch -d nomedabranch
Deleta a ramificação.

git branch
Lista todas as ramificações.
